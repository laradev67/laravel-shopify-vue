# Tailwind CSS preset for the Laravel framework

### Installing
```bash
$ npm install
```

### Setting up the new preset
```bash
$ php artisan preset tailwind
```

### Auth scaffolding
```bash
$ php artisan preset tailwind-auth
```

### Compiling assets
```bash
$ npm run dev
```

Enjoy!
